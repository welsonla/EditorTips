#####  Install 'code' Command
```
Shift+Command+P
Select -> Shell Command: Install 'code' command in PATH
then you can open VSCode in Terminal by type 'code'
```

##### Configuration

```
	//Turn on auto save
    "files.autoSave": "afterDelay",

    //Laravel Blade
    "emmet.triggerExpansionOnTab": true,
    "blade.format.enable": true,     

    //Beauty
    "beautify.language": {
        "js": {
          "type": [
            "javascript",
            "json",
            "jsonc"
          ],
          "filename": [
            ".jshintrc",
            ".jsbeautifyrc"
          ]
        },
        "css": [
          "css",
          "scss"
        ],
        "html": [
          "htm",
          "html",
          "blade"
        ]
      }
```
